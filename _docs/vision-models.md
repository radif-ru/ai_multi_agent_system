# Vision-модели для локального запуска

Рекомендации по выбору vision-моделей для Ollama (локальный запуск).

## Сравнение моделей

| Модель | Размер | Скорость | Качество | Use case |
|--------|--------|----------|----------|----------|
| moondream2 | ~1.6B | ⭐⭐⭐⭐⭐ | ⭐⭐ | Очень быстрая, минимум ресурсов, базовое описание картинок |
| llava-phi3 | ~4B | ⭐⭐⭐⭐ | ⭐⭐⭐ | Баланс скорости и качества, хороший выбор по умолчанию |
| gemma3:4b | ~4B | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | Нативно мультимодальная от Google, рекомендуется по умолчанию |
| minicpm-v | ~8B | ⭐⭐⭐ | ⭐⭐⭐⭐ | OCR (текст на картинках), хорошее качество |
| qwen2.5-vl:7b | ~7B | ⭐⭐ | ⭐⭐⭐⭐⭐ | Структурированный анализ: таблицы, диаграммы, документы |

## Рекомендации

**По умолчанию:** `gemma3:4b` — лучший баланс качества и скорости для большинства задач.

**Альтернативы:**
- Очень слабое железо → `moondream2`
- Нужен OCR → `minicpm-v`
- Сложные диаграммы/таблицы → `qwen2.5-vl:7b`

## Установка

```bash
ollama pull gemma3:4b
# или
ollama pull llava-phi3
```

## Конфигурация

В `.env`:
```bash
# Vision-модель по умолчанию
VISION_MODEL=gemma3:4b
```
